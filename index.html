SiView

An open-source 3D GDS viewer with a simple flow for Cadence integration. Explore IC layouts in three dimensions across processes and geometries.

View on GitHub Read updates on Substack

What it is

SiView renders GDSII layouts as interactive 3D geometry for faster design reviews, teaching, and debugging of complex analog and mixed-signal blocks.

Key goals

  • Works with standard .gds files
  • Cross-geometry and process agnostic
  • Simple hooks for Cadence flows (SKILL helper)
  • Lightweight and open source

Quick start

  1. Clone the repo: git clone https://github.com/gerhenry/SiView.git
  2. Open the example & run the viewer
  3. (Optional) Load the SKILL helper in Virtuoso to export a GDS and launch the viewer

; Example SKILL
; load("/path/to/icdGDS3D.il")
